# 薄い圏の考察 / Thin Category Notes

みどりりゅうによる「薄い圏の考察」シリーズのまとめです。

このリポジトリは、著者自身による**独自の考察**をまとめたものです。  
射を高々一つしか持たない薄い圏（thin category）を、対象間の二項演算として捉え直し、その性質を体系的に調べたものです。

## 概要

圏論において薄い圏を「前順序集合の一般化」として扱い、以下のテーマを考察しています：

- 薄い圏の新しい定義とその性質
- 局所的小圏から薄い圏への薄化関手（thinning functor）
- 括射関手（packed arrows functor）
- 薄い圏をターゲットとしたKan拡張、コエンド、エンド
- 上限・下限を通じた普遍的対象の扱い

本考察の特徴は、薄い圏特有の **「射の存在」だけで自然変換や普遍性を扱える**点に注目したことにあります。
